Introduction

This project reproduces and extends the Neural Monte Carlo Fluid Simulation paper from SIGGRAPH 2024. The original paper by Jain et al. presents a mesh-free fluid simulation method combining neural networks with Monte Carlo techniques for pressure solving.

My implementation focuses on Monte Carlo particle-based fluid dynamics with 300,000 particles, featuring multi-scale turbulence using 3-level Perlin noise, boundary condition enforcement for obstacle interaction, and density field voxelization via Gaussian kernel splatting.

Methods

Overview

The system implements a particle-based fluid simulator managing up to 300,000 active particles. Each particle carries position, velocity, lifetime, and density information. The simulation pipeline combines Monte Carlo methods for particle advection with multi-scale Perlin noise for turbulence generation.

Pipeline

  1. Particle System Initialization
    • Initialize 300,000 particle slots with attributes
    • Set up emission sources based on simulation mode
    • Configure 150×150×150 voxel grid for density field
  2. Per-Frame Update Loop
    • Generate 2,500 new particles per frame
    • Apply forces: turbulence, buoyancy, obstacle repulsion
    • Update positions and apply damping
    • Remove dead particles and voxelize active ones
Turbulence Model: Combines three octaves of Perlin noise with weights 1.0, 0.5, and 0.3. Time-evolve noise field: noise(pos, time) = perlin(pos + time × speed)

Three Modes

  1. Mode 1: Rising Smoke Plume (100 frames, 8-10 min)
  2. Mode 2: Colliding Smoke Rings (80 frames, 6-8 min)
  3. Mode 3: Obstacle Interaction (120 frames, 10-12 min)

Conclusion

Successfully demonstrates particle-based fluid simulation with Monte Carlo methods and multi-scale turbulence. Achieves visually plausible smoke behavior across three scenarios with reasonable performance.

Limitations: Simplified physics, no neural pressure solver, single-threaded implementation.
